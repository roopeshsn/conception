# Create a Activity Suggestion App

## Activity suggestion app to help bored users decide on what they want to randomly do on that day.



This app will be suggesting random activities that any bored user can do.

First ask the user if they are bored. If they are suggest them with an activity they can do. If they like it they can just go out and do it right away or plan it for some other time during the day. If they respond that they are not interested, prompt them with another activity.
